# ArtKech Design Studio — Print Production & Vendor Management

## 1. Purpose

ArtKech delivers both digital and physical design work. When projects include printed materials — business cards, brochures, packaging, signage, or large-format — we manage the entire production process from file preparation to final delivery. This SOP ensures print quality meets our standards, vendor relationships are maintained, and clients receive flawless physical products.

## 3. File Preparation Standards

### 3.1 Pre-Flight Checklist (Before Sending to Vendor)

| Check | Standard | Tool |
| :--- | :--- | :--- |
| Color mode | CMYK (no RGB elements remaining) | Illustrator: Edit → Assign Profile |
| Resolution | All raster elements ≥ 300 DPI | Illustrator: Links panel → check PPI |
| Bleed | 3mm on all sides (5mm for large format) | Document setup |
| Safe zone | Critical content 5mm from trim edge | Visual check |
| Fonts | Outlined (no live text in final print file) | Type → Create Outlines |
| Black | Rich black for large areas (C:40 M:30 Y:30 K:100) | Swatches check |
| Overprint | Black text set to overprint, no white overprint | Attributes panel |
| Spot colors | Pantone references noted if required | Swatches panel |
| Transparency | Flattened or saved as PDF/X-4 | Export settings |
| File format | PDF/X-4 (preferred) or packaged AI file | Export |

### 3.2 File Naming Convention

```
[Client]_[Piece]_[Size]_[Version]_PRINT.pdf

Examples:
CafeNomade_BusinessCard_85x55mm_v3_PRINT.pdf
RiadRoutes_Brochure_A4-trifold_v2_PRINT.pdf
ArtKech_Banner_3000x1000mm_v1_PRINT.pdf
```

### 3.3 Delivery to Vendor

| Method | When to Use |
| :--- | :--- |
| WeTransfer / Google Drive link | Standard files (< 2GB) |
| USB drive (hand-delivered) | Large format files, sensitive projects |
| Vendor's FTP/upload portal | If vendor provides one |
| Email attachment | Only for files < 10MB (proofs, simple jobs) |

Always include a production brief document with:
- Quantity
- Paper/material specification
- Finishing (lamination, UV, foil, die-cut)
- Delivery address and deadline
- Contact person for questions
- Special instructions

## 4. Proofing Process

### 4.1 Proof Types

| Proof Type | When Required | Cost | Turnaround |
| :--- | :--- | :--- | :--- |
| Digital proof (PDF) | All jobs | Free | Same day |
| Color proof (Epson/inkjet) | Color-critical jobs (packaging, brand materials) | 200-500 MAD | 1-2 days |
| Physical sample (actual material) | New materials, first-time vendor, luxury finishes | 500-2,000 MAD | 3-5 days |
| Press proof (on-press approval) | High-volume runs (> 5,000 units) | Included in setup | Scheduled |

### 4.2 Proof Review Checklist

| Element | What to Check |
| :--- | :--- |
| Content accuracy | All text correct, no typos, phone numbers verified |
| Color fidelity | Colors match brand guidelines (compare to Pantone swatch) |
| Image quality | No pixelation, banding, or color shifts |
| Trim & bleed | Content properly positioned, bleed extends correctly |
| Finishing marks | Die-cut lines, fold marks, perforation lines correct |
| Paper/material | Correct stock, weight, and finish specified |
| Quantity | Confirmed and matches order |

### 4.3 Proof Approval

- Designer reviews proof first (technical check)
- Account Manager reviews (content accuracy)
- Client reviews (if client is involved in print decisions)
- Written approval required before production begins ("Bon à tirer" / BAT)
- After BAT approval: any errors are client's responsibility (we document this clearly)

## 5. Quality Control on Delivery

### 5.1 Inspection Protocol

When printed materials arrive at the studio:

| Step | Action | Criteria |
| :--- | :--- | :--- |
| 1 | Visual inspection (random sample of 10%) | No visible defects, smudges, misalignment |
| 2 | Color check (compare to approved proof) | Color within acceptable tolerance |
| 3 | Trim check (measure dimensions) | Within ±0.5mm of specified size |
| 4 | Finishing check | Lamination smooth, foil crisp, die-cut clean |
| 5 | Quantity count (for orders > 500 units) | Within ±2% of ordered quantity |
| 6 | Packaging condition | No damage, corners protected, shrink-wrapped if specified |

### 5.2 Defect Handling

| Defect Severity | Examples | Action |
| :--- | :--- | :--- |
| Critical (unusable) | Wrong content, major color shift, wrong material | Full reprint at vendor's cost, deadline extended |
| Major (noticeable) | Slight misregistration, minor color variation, scuff marks | Negotiate partial reprint or discount |
| Minor (acceptable) | Barely visible variation within industry tolerance | Accept with documentation, note for future orders |

## 6. Client Delivery

### 6.1 Delivery Options

| Method | When | Cost to Client |
| :--- | :--- | :--- |
| Client picks up at studio | Local clients, small orders | Free |
| Courier delivery (Marrakech) | Standard | 50-100 MAD |
| Courier delivery (national) | Casablanca, Rabat, Tangier, etc. | 150-300 MAD |
| International shipping | Foreign clients | Actual cost + 200 MAD handling |
| Direct to venue/event | Event materials, signage | Actual cost + coordination fee |

### 6.2 Delivery Presentation

- Materials are never delivered in vendor's generic packaging
- Repackaged in ArtKech branded tissue paper + branded sticker seal
- Include a small card: "Crafted with care by ArtKech Design Studio"
- Business cards delivered in a branded card holder (for orders > 250 cards)
- Large format items: photos taken of installed signage, sent to client

## 7. Cost Management

### 7.1 Markup Policy

| Order Value (vendor cost) | ArtKech Markup | Justification |
| :--- | :--- | :--- |
| < 1,000 MAD | 40% | File prep, proofing, QC, delivery coordination |
| 1,000 – 5,000 MAD | 30% | Same as above, economies of scale |
| 5,000 – 20,000 MAD | 25% | Larger orders, less per-unit management |
| > 20,000 MAD | 20% | Volume discount passed partially to client |

### 7.2 Quote Process

1. Receive print specifications from designer/account manager
2. Request quotes from 2-3 vendors (minimum 2 for orders > 5,000 MAD)
3. Compare on: price, quality, timeline, and reliability
4. Select vendor and add ArtKech markup
5. Present quote to client with timeline
6. Upon approval: collect payment (100% upfront for print, or 50% deposit for orders > 10,000 MAD)

## 8. Sustainability Practices

| Practice | Standard |
| :--- | :--- |
| Paper sourcing | Prefer FSC-certified or recycled stock when available |
| Ink | Vegetable-based inks preferred for offset |
| Waste reduction | Order exact quantities + 5% overs (not 20%) |
| Digital alternatives | Always offer digital version alongside print |
| Local vendors | Prioritize Marrakech-based vendors (reduce transport emissions) |
| Packaging | Minimal packaging, recyclable materials only |
