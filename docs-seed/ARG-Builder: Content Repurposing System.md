# ARG-Builder: Content Repurposing System

## Executive Summary

This document defines ARG-Builder's content repurposing methodology — the systematic approach to maximizing the ROI of every piece of content by transforming it into multiple formats across multiple channels. A single pillar piece of content should generate 15–25 derivative assets, reducing production costs by 60–70% while maintaining consistent messaging.

## Quality Standards

### Repurposing Rules

| Rule | Implementation |
|------|---------------|
| Never copy-paste | Each derivative must be native to its channel |
| Maintain core message | Key insight must survive in every format |
| Optimize for channel | Length, format, tone must match platform norms |
| Add new value | Each derivative should offer something the original didn't |
| Update data | Refresh statistics and examples for each new format |
| Credit sources | Always link back to original when appropriate |
| Track performance | Measure each derivative independently |
