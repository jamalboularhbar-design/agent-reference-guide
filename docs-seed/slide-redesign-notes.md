# Slide Redesign Notes

## Design System
- Background: #0f0f0f (near-black, not brown)
- Accent: #e8a04e (amber gold)
- Text primary: #f5f5f5 (near-white)
- Text secondary: #999999 (medium gray)
- Text muted: #666666 (dark gray)
- Surface: #1a1a1a (subtle dark card)
- Border/divider: #2a2a2a
- Success: #4ade80 (green)

## Typography
- Headings: Space Grotesk, 700 weight
- Body: Inter, 400 weight
- Title slide: 64px heading
- Content slides: 36px heading, 15px subtitle, 13-14px body

## Layout Principles
- Generous padding: 64px top, 56px sides
- Max content width: 960px
- Consistent left-aligned headings with thin amber underline accent
- No border-left accents on cards — use subtle background difference only
- No cramped grids — max 3 columns for content, 4 for small metrics
- Breathing room between sections: 32px minimum
- Cards: no borders, just subtle bg difference (#1a1a1a on #0f0f0f)
