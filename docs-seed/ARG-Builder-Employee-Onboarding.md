# ARG-Builder: Employee Onboarding Playbook

## 90-Day Framework for Integrating New Hires into High-Performance Teams

---

## 1. Executive Summary

The first 90 days of employment determine whether a new hire becomes a high-performer or a regrettable attrition. Companies with structured onboarding programs achieve 82% higher retention and 70% faster time-to-productivity. For ARG-Builder's rapid growth phase, a repeatable, high-quality onboarding experience ensures every new team member contributes quickly, feels connected to the mission, and builds lasting relationships. This playbook defines the complete onboarding system.

---

## 2. Onboarding Philosophy

### 2.1 Core Principles

| Principle | Description | Implementation |
|-----------|-------------|---------------|
| Belonging first | People perform when they feel they belong | Day 1 warmth, buddy system, team rituals |
| Context before tasks | Understand the "why" before the "what" | Mission, strategy, and customer immersion first |
| Progressive complexity | Start simple, increase responsibility gradually | 30-60-90 day milestones |
| Two-way feedback | New hires have fresh eyes — capture their insights | Structured feedback at 30, 60, 90 days |
| Manager ownership | Managers own onboarding success, HR enables | Manager checklist, accountability |
| Self-directed learning | Provide resources, trust adults to learn | Learning library, documentation access |
| Early wins | Quick victories build confidence and credibility | First-week deliverable assigned |

### 2.2 Onboarding Outcomes

| Timeframe | Outcome | Measurement |
|-----------|---------|-------------|
| Day 1 | "I feel welcome and know what to expect" | Day 1 survey |
| Week 1 | "I understand the company, product, and my role" | Week 1 check-in |
| Day 30 | "I've delivered my first meaningful contribution" | Manager assessment |
| Day 60 | "I'm operating independently in my core responsibilities" | Manager assessment |
| Day 90 | "I'm fully productive and integrated into the team" | Performance baseline |

| Retention correlation analysis | Quarterly | ROI validation |

### 9.3 Common Failure Modes

| Failure | Signal | Prevention |
|---------|--------|-----------|
| Information overload | New hire overwhelmed, disengaged | Progressive disclosure, pace learning |
| Isolation | No buddy, no team connection | Mandatory buddy, social events |
| Unclear expectations | New hire unsure what success looks like | Written 30-60-90 plan |
| Manager absence | Manager too busy for 1:1s | Manager accountability, backup |
| Tool/access delays | Can't work due to missing access | Pre-boarding checklist |
| No early wins | New hire feels unproductive | Assign quick-win task Week 1 |
| Cultural disconnect | New hire doesn't understand norms | Culture sessions, buddy guidance |

## 10. Remote/Hybrid Considerations

### 10.1 Remote Onboarding Adaptations

| Challenge | Solution |
|-----------|---------|
| Lack of physical presence | Video-on for all meetings, virtual office hours |
| Isolation | Daily buddy check-ins, virtual coffee chats |
| Tool setup | Pre-configured laptop shipped, IT support on standby |
| Relationship building | Scheduled 1:1s with 10+ team members in first 2 weeks |
| Culture absorption | Recorded culture sessions, virtual team events |
| Informal learning | Slack channels for questions, recorded meetings |
| Time zone challenges | Overlap hours defined, async-first documentation |

### 10.2 Hybrid Onboarding Best Practices

| Practice | Implementation |
|----------|---------------|
| First week in-office | Require in-person for Week 1 (if possible) |
| In-person milestones | 30-day and 90-day reviews in-person |
| Team days | Align new hire start with team in-office days |
| Documentation-first | All processes documented, not just tribal knowledge |
| Async updates | Daily written updates in Slack during ramp |
| Virtual social | Weekly virtual team social (optional but encouraged) |
