# ArtKech Design Studio — Financial Management & Invoicing

## 1. Purpose

Creative talent means nothing without financial sustainability. This SOP defines how ArtKech manages its finances: invoicing clients, tracking expenses, managing cash flow, budgeting, and ensuring the studio remains profitable while investing in growth. Every team member who touches money (quotes, expenses, invoices) must follow these procedures.

## 2. Revenue Streams

| Stream | % of Revenue | Payment Cycle | Predictability |
| :--- | :--- | :--- | :--- |
| Project-based work (one-time) | 45% | Per project (deposit + final) | Variable |
| Monthly retainers (social media, ongoing design) | 35% | Monthly recurring | Highly predictable |
| Photography & production | 10% | Per shoot/project | Variable |
| Consulting & strategy | 5% | Per engagement | Variable |
| Workshops & training | 5% | Per event | Seasonal |

## 3. Invoicing Process

### 3.1 Invoice Types

| Type | When Issued | Payment Terms |
| :--- | :--- | :--- |
| Deposit invoice | Upon contract signing | Due immediately (before work starts) |
| Progress invoice | At agreed milestone | Net 7 days |
| Final invoice | Upon project completion/delivery | Net 7 days (files held until paid) |
| Retainer invoice | 1st of each month | Net 15 days |
| Rush fee invoice | When rush work is confirmed | Due immediately |
| Change order invoice | When scope change is approved | Net 7 days |

```
AK-[YEAR]-[SEQUENTIAL NUMBER]

Examples:
AK-2026-0001 (first invoice of 2026)
AK-2026-0047 (47th invoice of 2026)
```

| Event | Action | Responsible |
| :--- | :--- | :--- |

### 4.2 Cash Flow Protection Rules

| Rule | Standard |
| :--- | :--- |
| Never start work without deposit | 50% minimum for projects > 10,000 MAD |
| Never deliver final files without full payment | Files held in escrow |
| Retainer clients: suspend service after 15 days overdue | Resume upon payment |
| New clients: 100% upfront for first project < 10,000 MAD | Build trust first |
| International clients: full payment upfront | Wire transfer before work begins |

## 7. Project Profitability Tracking

### 7.1 Per-Project P&L

| Line Item | Calculation |
| :--- | :--- |
| Revenue | Total invoiced to client (HT) |
| - Direct labor cost | Hours tracked × internal hourly rate |
| - Freelancer costs | Invoices from freelancers on this project |
| - Direct expenses | Stock photos, printing, travel, materials |
| = Gross margin | Revenue - all direct costs |
| Gross margin % | Gross margin / Revenue × 100 |

### 7.2 Profitability Targets

| Project Type | Target Gross Margin | Minimum Acceptable |
| :--- | :--- | :--- |
| Brand identity | 65% | 50% |
| Print design | 60% | 45% |
| Social media retainer | 55% | 40% |
| Web design | 60% | 45% |
| Photography | 50% | 35% |
| Rush projects | 70% | 60% |

### 7.3 Post-Project Financial Review

After every project > 15,000 MAD:
1. Calculate actual hours vs. estimated hours
2. Calculate actual margin vs. target margin
3. Identify what caused overruns (if any)
4. Document lessons for future estimation
5. Flag if client should be re-priced for future work
