# ARG-Builder: Customer Education Program

## Executive Summary

This document defines ARG-Builder's customer education program — the academy structure, certification programs, help center architecture, community building strategy, and content delivery systems that ensure customers achieve maximum value from the platform. Education is not a support cost — it is a growth lever. Educated customers adopt faster, expand more, churn less, and become advocates.

### Course Structure

Each course follows a consistent structure:

| Component | Duration | Format | Purpose |
|-----------|----------|--------|---------|
| Introduction | 2 min | Video | Context, learning objectives |
| Concept explanation | 5–8 min | Video + text | Teach the concept |
| Guided walkthrough | 5–10 min | Interactive demo | Show how it works |
| Practice exercise | 5–10 min | Hands-on sandbox | Apply the learning |
| Knowledge check | 3 min | Quiz (3–5 questions) | Validate understanding |
| Summary + next steps | 2 min | Text | Recap, link to next course |

## Content Production Workflow

### Monthly Content Calendar

| Week | Content Type | Quantity | Owner |
|------|-------------|----------|-------|

| Week 2 | Video tutorials | 2 | Education team |

### Content Quality Standards

| Standard | Requirement |
|----------|-------------|
| Accuracy | All steps verified in current product version |
| Clarity | 8th-grade reading level, no jargon without definition |
| Visuals | Screenshots for every 3+ step procedure |
| Accessibility | Alt text, captions, transcript for video |
| Freshness | Reviewed and updated within 90 days of product changes |
| Feedback | Helpfulness rating visible, feedback mechanism on every page |
