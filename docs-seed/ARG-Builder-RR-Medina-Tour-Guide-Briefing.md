# Riad & Routes — Medina Tour Guide Briefing & Quality Standards

## 1. Purpose

Guided tours through Marrakech's medina are one of our highest-rated experiences. This document defines the briefing process for guides, quality standards for each tour type, safety protocols, and guest satisfaction benchmarks. Whether using our in-house guides or vetted freelancers, every tour must deliver the Riad & Routes standard of excellence.

## 3. Guide Requirements

### 3.1 In-House Guides (2 full-time)

| Requirement | Standard |
| :--- | :--- |
| Licensing | Official Moroccan tourism guide license (carte professionnelle) |
| Languages | Minimum 3 fluent (Arabic + French + English or Spanish) |
| Experience | 5+ years guiding in Marrakech medina |
| Knowledge test | Annual assessment: history, architecture, culture, current events |
| First aid | Current certification (renewed annually) |
| Appearance | Clean, smart-casual dress code, R&R branded polo or pin |
| Phone | Fully charged, WhatsApp available, Google Maps offline downloaded |

### 3.2 Freelance Guide Vetting

Before any freelance guide is approved:
1. License verification (copy on file)
2. Reference check (minimum 2 previous employers or 50+ TripAdvisor reviews)
3. Shadow tour (accompanies our in-house guide for 1 full tour, assessed)
4. Trial tour (leads a tour with R&R staff member as "guest," assessed)
5. Background check (police clearance certificate)
6. Contract signed (service standards, cancellation policy, payment terms)

## 4. Pre-Tour Briefing (Day Before)

The Guest Relations Manager briefs the guide by 18:00 the evening before:

| Information | Source | Why It Matters |
| :--- | :--- | :--- |
| Guest names and nationalities | PMS booking | Personalization, language choice |
| Number of guests + ages | PMS booking | Pace adjustment, child-friendly stops |
| Physical limitations | Guest profile / pre-arrival questionnaire | Route modification (avoid stairs, long walks) |
| Interests noted | Guest profile / concierge notes | Emphasize relevant stops (art, food, history) |
| Dietary restrictions (food tour) | Guest profile | Avoid allergens at tasting stops |
| Previous Marrakech visits | Concierge check | Avoid repeating what they've already seen |
| Special requests | WhatsApp / concierge | Photography focus, shopping assistance, etc. |
| Weather forecast | Weather app | Shade route if hot, covered stops if rain risk |

## 5. Tour Execution Standards

### 5.1 Meeting Point & Departure

| Step | Standard |
| :--- | :--- |
| Guide arrives at riad | 15 minutes before departure time |
| Introduction | Warm greeting by name, brief self-introduction, tour overview |
| Comfort check | "Do you have comfortable shoes? Water? Sunscreen? Hat?" |
| Safety briefing | "Stay close in crowded areas, keep valuables secure, watch for motorbikes" |
| Pace setting | "Please let me know if I'm walking too fast or if you need a break at any time" |
| Departure | Leave on time (±5 minutes). If guest is late, wait maximum 10 minutes then call. |

### 5.2 During the Tour

| Standard | Description |
| :--- | :--- |
| Storytelling over lecturing | Share anecdotes, personal stories, local legends — not textbook history |
| Engagement | Ask questions, invite observations, encourage photos at key spots |
| Pace | Stop every 20-25 minutes for a rest, photo opportunity, or tasting |
| Hydration | Offer water at least twice during the tour (guide carries extra bottles) |
| Shopping pressure | NEVER take guests to shops for commission without their explicit request |
| Flexibility | If guests show strong interest in something, extend that stop and shorten another |
| Local interactions | Introduce guests to artisans, shopkeepers, locals (pre-arranged, genuine) |
| Photography | Offer to take group photos at scenic spots (without being asked) |

### 5.3 Tour Conclusion

| Step | Standard |
| :--- | :--- |
| Return to riad (or agreed endpoint) | On time, within 15 minutes of scheduled end |
| Summary | Brief recap of highlights, answer final questions |
| Recommendations | Suggest related experiences for remaining days |
| Farewell | Warm goodbye, hand business card, "I'm available if you need anything during your stay" |
| Feedback | "I hope you enjoyed the tour. If you have a moment, a review on TripAdvisor would mean a lot." |

## 6. Safety Protocols

### 6.1 Risk Assessment by Tour Type

| Risk | Mitigation |
| :--- | :--- |
| Motorbike traffic in narrow alleys | Guide walks on traffic side, warns at intersections, uses hand signals |
| Pickpockets in crowded souks | Brief guests before entering, suggest cross-body bags, stay close |
| Heat exhaustion (summer) | Shade route, frequent water stops, shortened duration if above 40°C |
| Getting lost (guest separated) | Give guests the riad's WhatsApp number and address card before departure |
| Aggressive vendors | Guide intervenes politely, redirects conversation, keeps group moving |
| Food safety (food tours) | Only visit pre-vetted vendors with hygiene standards, avoid raw items |
| Allergic reactions | Guide carries antihistamine, knows nearest pharmacy location |

## 7. Quality Control

### 7.1 Mystery Guest Audits

Quarterly, an unannounced "mystery guest" (staff member or hired auditor) joins a tour and evaluates:

| Criteria | Weight | Pass Score |
| :--- | :--- | :--- |
| Punctuality and preparation | 10% | 90% |
| Knowledge accuracy | 20% | 85% |
| Storytelling and engagement | 25% | 80% |
| Guest interaction and personalization | 20% | 85% |
| Safety awareness | 15% | 95% |
| Professional appearance | 10% | 90% |

### 7.2 Guest Feedback Analysis

After every tour, guests receive a 3-question SMS/WhatsApp survey:
1. "How would you rate your tour experience? (1-5)"
2. "What was the highlight?"
3. "Is there anything we could improve?"

Results are compiled monthly and shared with guides.

### 7.3 Guide Performance Metrics

| KPI | Target | Action if Below |
| :--- | :--- | :--- |
| Average guest rating | > 4.8/5.0 | Coaching session with Guest Relations Manager |
| On-time departure rate | 100% | Verbal warning after 2 late starts |
| Safety incident rate | 0 per quarter | Immediate review and retraining |
| Rebooking rate (guest books another tour) | > 40% | Upselling training |
| TripAdvisor mentions (by name) | 2+ per month | Recognition bonus |
