# ARG-Builder: Pricing Page Optimization

## Executive Summary

This document defines ARG-Builder's pricing page strategy — the design, copy, psychology, and conversion optimization framework for the most critical page on the marketing website. The pricing page is where intent converts to action; it must communicate value clearly, reduce friction, and guide prospects toward the right plan with confidence.

## Page Design Framework

### Above-the-Fold Structure

| Section | Content | Purpose |
|---------|---------|---------|
| Headline | "Simple pricing that scales with you" | Set expectations, reduce anxiety |
| Subheadline | "Start free. Upgrade when you're ready. No surprises." | Remove risk, build trust |
| Toggle | Monthly / Annual (show savings) | Anchor on annual, show value |
| Plan cards | 3 columns with clear hierarchy | Enable comparison |
| Social proof | "Trusted by 200+ companies" + logos | Build credibility |

### Plan Card Anatomy

| Element | Purpose | Best Practice |
|---------|---------|--------------|
| Plan name | Quick identification | Descriptive, not clever |
| Price | Anchor value | Large font, monthly price |
| Per unit | Clarify what's included | "per month, billed annually" |
| Description | 1-line value prop per tier | Who this is for |
| Feature list | What's included | 5–8 most important features |
| CTA button | Drive action | Contrasting color, action verb |
| Popular badge | Guide choice | On recommended plan (Professional) |

## Conversion Psychology

### Pricing Psychology Tactics

| Tactic | Implementation | Expected Impact |
|--------|---------------|-----------------|
| Anchoring | Show Enterprise first (right to left) or highlight Professional | +15% Professional selection |
| Decoy effect | Starter exists to make Professional look like better value | +20% Professional conversion |
| Loss aversion | "Save $12,000/year" on annual toggle | +30% annual billing selection |
| Social proof | "Most popular" badge on Professional | +25% Professional selection |
| Urgency | "Lock in current pricing" for early adopters | +10% conversion rate |
| Risk reversal | "30-day money-back guarantee" | +15% trial starts |
| Round numbers | $2,000 not $1,997 (B2B = trust, not tricks) | Higher perceived quality |

### Objection Handling (On-Page)

| Objection | Solution | Placement |
|-----------|----------|-----------|
| "Too expensive" | ROI calculator showing 10x return | Below pricing cards |
| "Not sure which plan" | Interactive quiz: "Which plan is right for you?" | Above pricing cards |
| "Need to talk to team" | "Share this page" button + comparison PDF download | Below CTA |
| "What if it doesn't work?" | 30-day guarantee + case studies | Below pricing cards |
| "Hidden costs?" | "No hidden fees. No setup costs. Cancel anytime." | Below toggle |
| "Need custom solution" | "Contact us for custom pricing" + form | Enterprise card |
