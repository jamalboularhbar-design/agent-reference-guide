# Riad & Routes — Night Audit & Security Procedures

## 1. Purpose

This SOP covers overnight operations from 22:00 to 06:00, including the night audit financial reconciliation, security protocols, emergency procedures, and guest safety measures. The night shift is the most vulnerable period for any property — fewer staff, sleeping guests, and reduced visibility require heightened vigilance and clear protocols.

## 2. Night Shift Staffing

| Role | Hours | Responsibilities |
| :--- | :--- | :--- |
| Night Manager/Auditor | 22:00 – 06:00 | Financial audit, guest requests, security oversight, emergency response |
| Night Security | 22:00 – 06:00 | Perimeter patrol, door monitoring, CCTV surveillance, incident response |
| Night Porter | 22:00 – 06:00 | Late arrivals, luggage, room service (cold items), laundry processing |

Minimum staffing: 2 persons on-site at all times (Night Manager + Security). Night Porter is scheduled based on occupancy (>60% = required).

## 3. Night Audit Process (22:30 – 00:30)

### 3.1 Revenue Reconciliation

| Task | Action | Verification |
| :--- | :--- | :--- |
| Room charges | Verify all occupied rooms are correctly charged at the correct rate | Cross-check PMS vs booking confirmation |
| F&B charges | Match all restaurant/bar bills to room folios or cash receipts | POS system vs PMS posting |
| Spa charges | Confirm all treatments are posted to correct rooms | Spa booking sheet vs PMS |
| Tour/experience charges | Verify all tour payments (cash collected, card charged, or room charge) | Tour log vs PMS |
| Cash reconciliation | Count all cash drawers (front desk, restaurant, spa) | Physical count vs system total |
| Card settlements | Batch close all card terminals, verify totals match PMS | Terminal report vs PMS |
| Petty cash | Verify petty cash balance matches log (receipts + remaining cash = float) | Physical count |

### 3.2 Occupancy & Statistics

The Night Auditor produces the daily statistics report:

| Metric | Calculation |
| :--- | :--- |
| Occupancy % | Rooms sold / Total rooms available × 100 |
| ADR | Total room revenue / Rooms sold |
| RevPAR | Total room revenue / Total rooms available |
| Total revenue | Room + F&B + Spa + Tours + Minibar + Other |
| No-shows | Confirmed bookings that did not arrive |
| Walk-ins | Guests who arrived without reservation |
| Complimentary rooms | Rooms provided at zero charge (staff, owner, comp) |

### 3.3 System Tasks

- Roll the date in PMS (advance to next business day)
- Generate and email the daily report to GM, Revenue Manager, and Finance
- Back up PMS data (automated but verify completion)
- Print next day's arrival list and place on front desk
- Update the occupancy board for morning shift briefing

## 4. Security Protocols

### 4.1 Perimeter Patrol Schedule

| Time | Route | Duration | Focus Areas |
| :--- | :--- | :--- | :--- |
| 22:30 | Full property exterior + rooftop | 20 min | Gate locks, street-facing windows, rooftop access |
| 00:00 | Interior corridors + courtyard | 15 min | Room doors closed, no unauthorized persons, fire exits clear |
| 02:00 | Full property exterior + parking | 20 min | Vehicle security, perimeter walls, garden areas |
| 04:00 | Interior corridors + service areas | 15 min | Kitchen locked, storage rooms secure, no water leaks |
| 05:30 | Final exterior check before dawn | 15 min | Prepare for morning staff arrival, unlock service entrance |

### 4.2 Access Control

| Access Point | Night Protocol |
| :--- | :--- |
| Main entrance (guest door) | Locked at 23:00. Guests ring bell for entry. Security verifies via camera before opening. |
| Service entrance (staff door) | Locked at 22:00. Only Night Manager has key. |
| Rooftop access | Locked at 22:00. Available on request (romantic dinner, stargazing). |
| Kitchen | Locked at 23:00. Night Porter has access for cold items only (no cooking). |
| Storage rooms | Locked at 22:00. Night Manager access only. |
| Safe deposit (office) | Double-locked. Two keys required (Night Manager + sealed envelope in safe). |

### 4.3 CCTV Monitoring

- 12 cameras cover: main entrance, service entrance, corridors, courtyard, parking, rooftop access, reception
- Cameras are NOT placed in: guest rooms, bathrooms, private terraces, spa areas
- Night Security checks all camera feeds every 30 minutes
- Recordings are stored for 30 days, then overwritten
- Any suspicious activity is screenshot, logged, and reported to GM by 07:00

### 4.4 Key Management

- Master key (opens all rooms): held by Night Manager only, signed out at shift start, returned at shift end
- Emergency key box: sealed box in reception containing master key — only broken open in emergency (fire, medical)
- Guest room keys: if a guest loses their key at night, verify identity (name, passport number, booking reference) before issuing replacement
- All key movements are logged in the key register

## 5. Emergency Procedures

### 5.1 Fire

1. Activate fire alarm (manual pull station in reception)
2. Call fire department: 15 (Morocco emergency) or 0524-XXX-XXX (local fire station)
3. Night Manager takes guest list and begins room-by-room evacuation (knock loudly, announce "Fire evacuation")
4. Security opens all fire exits and guides guests to assembly point (courtyard or street)
5. Account for all guests using the occupancy list
6. Do NOT attempt to fight the fire unless it is small and contained (use extinguisher only if safe)
7. Meet fire department at main entrance, provide building layout

### 5.2 Medical Emergency

1. Assess the situation (conscious? breathing? bleeding?)
2. Call ambulance: 15 or 0524-XXX-XXX (Clinique Internationale Marrakech — 10 min away)
3. Administer first aid if trained (first aid kit in reception + AED device on wall)
4. Do NOT move the person unless they are in immediate danger
5. Night Manager stays with the guest; Security meets the ambulance at the entrance
6. Contact guest's emergency contact (from registration card)
7. Log the incident in detail (time, symptoms, actions taken, responders)

### 5.3 Intruder / Security Threat

1. Do NOT confront the intruder directly
2. Lock the reception area, secure the cash drawer
3. Call police: 19 (Morocco police) or 0524-XXX-XXX (local commissariat)
4. If guests are at risk, activate the silent alarm (button under reception desk)
5. Monitor intruder via CCTV, provide real-time updates to police
6. After incident: full written report, CCTV footage preserved, GM notified immediately

### 5.4 Power Outage

1. Generator activates automatically within 30 seconds (covers: emergency lighting, reception, security cameras, refrigeration)
2. Night Manager checks that generator is running (listen for engine, verify lights)
3. If generator fails: deploy battery-powered emergency lights in corridors
4. Inform any awake guests that the situation is under control
5. Call electrician if outage exceeds 30 minutes or is property-specific (not city-wide)
6. Log outage duration and cause for maintenance follow-up

## 6. Late Arrival & Early Departure Handling

### 6.1 Late Arrivals (After 22:00)

- Night Porter waits at reception for expected late arrivals (list provided by evening shift)
- Abbreviated check-in: collect passport photo, hand over key, brief room orientation
- Welcome plate pre-set in room (fruit, water, pastries)
- Full registration completed next morning

### 6.2 Early Departures (Before 06:00)

- Guest notifies front desk the evening before (or via WhatsApp)
- Bill is prepared and slipped under the door the night before for review
- Night Porter arranges luggage assistance and calls taxi/transfer
- Express check-out: guest leaves key at reception, bill charged to card on file
- Farewell is warm but brief (respect the early hour)

## 7. Night Audit Reporting

The Night Auditor produces and distributes the following by 06:00:

| Report | Recipients | Content |
| :--- | :--- | :--- |
| Daily Revenue Summary | GM, Finance, Revenue Manager | All revenue figures, occupancy stats, variances |
| Cash Reconciliation | Finance Manager | Cash count, discrepancies, explanation |
| Incident Report (if any) | GM, Security Manager | Details of any security/safety/guest incidents |
| Arrival/Departure List | Front Desk (morning shift) | Today's expected arrivals, departures, special notes |
| Maintenance Log | Maintenance Manager | Any issues discovered during patrols (leaks, broken items) |

| :--- | :--- | :--- |

| Patrol completion | 100% of scheduled patrols | Patrol log timestamps |
| Late arrival handling time | < 10 minutes (arrival to room) | PMS timestamp |
| Incident response time | < 3 minutes | Incident log |
| Report delivery | By 06:00 every day | Email timestamp |
