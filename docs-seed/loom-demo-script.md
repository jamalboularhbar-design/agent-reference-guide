# ARG Builder — 60-Second Loom Demo Script

**Duration:** 60 seconds  
**Format:** Screen recording with voiceover  
**Audience:** Mid-market Ops leaders, VPs, and team leads  

---

## Pre-Recording Setup

1. Open ARG Builder at https://www.argbuilder.io
2. Log in as admin (ensure demo data is loaded)
3. Set browser to 1920x1080, dark mode
4. Close all other tabs and notifications
5. Have the Knowledge Graph page ready in a second tab

---

## Tips for Recording

- Speak at a natural pace — don't rush
- Move the cursor slowly and deliberately
- Pause 0.5s before each click so viewers can follow
- Use keyboard shortcuts (Cmd+K) for a polished feel
- Record 2-3 takes and pick the smoothest one
