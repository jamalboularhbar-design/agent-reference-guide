## Vertical-Specific Strategies

### Professional Services

| Element | Strategy |
|---------|----------|
| **ICP** | Consulting firms (50–500 employees), law firms (20–200 attorneys), accounting firms (Big 4 to mid-tier) |
| **Key pain** | Knowledge trapped in senior partners' heads, inconsistent service delivery, onboarding new associates takes 6–12 months |
| **Product adaptation** | Engagement playbooks, methodology templates, client-facing deliverable standards |
| **Pricing** | Premium tier ($8K–$15K/month) — high value per seat |
| **Channel** | Industry conferences (ALM, Consulting Magazine), partner with practice management tools |
| **Proof points needed** | "Reduced associate ramp time by 40%" / "Standardized delivery across 12 offices" |

### Healthcare

| Element | Strategy |
|---------|----------|
| **ICP** | Hospital systems (5–50 facilities), specialty clinics, health tech companies |
| **Key pain** | Clinical protocol adherence, staff onboarding in high-turnover environment, regulatory compliance documentation |
| **Product adaptation** | HIPAA-compliant deployment, clinical workflow templates, compliance tracking |
| **Pricing** | Enterprise tier ($10K–$25K/month) — compliance premium |
| **Channel** | HIMSS, health system CIOs, EHR integration partners |
| **Proof points needed** | "Reduced protocol deviation by 35%" / "Cut nursing onboarding from 12 weeks to 6" |
| **Compliance** | HIPAA BAA required, SOC 2 + HITRUST, data residency |

### Financial Services

| Element | Strategy |
|---------|----------|
| **ICP** | Regional banks (50–500 branches), insurance carriers, fintech companies (100–1,000 employees) |
| **Key pain** | Regulatory compliance procedures, inconsistent customer service across branches, M&A integration |
| **Product adaptation** | Compliance workflow templates, audit trail, role-based access, regulatory update integration |
| **Pricing** | Enterprise tier ($10K–$20K/month) — regulatory premium |
| **Channel** | Banking conferences (BAI), fintech partnerships, compliance consultants |
| **Proof points needed** | "Passed OCC exam with zero findings" / "Unified procedures across 200 branches post-merger" |
| **Compliance** | SOC 2 Type II, SOX controls, GLBA, state banking regulations |

### Technology / SaaS

| Element | Strategy |
|---------|----------|
| **ICP** | SaaS companies (50–500 employees), digital agencies, IT services firms |
| **Key pain** | Scaling tribal knowledge, onboarding engineers/PMs quickly, maintaining consistency across remote teams |
| **Product adaptation** | Engineering playbooks, product management frameworks, API documentation integration |
| **Pricing** | Professional tier ($3K–$8K/month) — volume play |
| **Channel** | Product Hunt, SaaStr, developer communities, integration marketplace |
| **Proof points needed** | "Reduced new engineer ramp from 3 months to 3 weeks" / "Standardized product process across 8 squads" |
