# ArtKech Design Studio — Photography Shoot Planning & Execution

## 1. Purpose

Photography is both a standalone service and a critical input to design projects. Whether shooting product photography for a restaurant client, capturing a riad's interiors, or documenting a brand launch event, every shoot must be planned meticulously to deliver maximum value within tight timelines and budgets. This SOP covers the complete workflow from brief to final delivery.

## 3. Pre-Shoot Planning

### 3.1 Planning Timeline

| Days Before Shoot | Action | Responsible |
| :--- | :--- | :--- |
| 14 days | Confirm shoot date, location, and objectives | Account Manager |
| 10 days | Create shot list and mood board | Photographer + Creative Director |
| 7 days | Scout location (if unfamiliar) | Photographer |
| 7 days | Confirm props, styling, and wardrobe needs | Photographer + Stylist |
| 5 days | Share shot list with client for approval | Account Manager |
| 3 days | Confirm all logistics (access, parking, power, timing) | Studio Coordinator |
| 1 day | Charge batteries, format cards, pack gear, test equipment | Photographer |
| Day of | Arrive 30 min early for setup | Photographer |

### 3.2 Shot List Template

| Shot # | Description | Angle/Composition | Lighting | Props/Styling | Priority |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 1 | Hero shot — signature dish on marble surface | Overhead (flat lay) | Natural + reflector | Fresh herbs, linen napkin, vintage cutlery | Must-have |
| 2 | Dish detail — texture close-up | 45° angle, shallow DOF | Side light (window) | Minimal — focus on food | Must-have |
| 3 | Ambiance — dining room with guests | Wide angle, eye level | Available light | Real guests (with permission) | Nice-to-have |
| 4 | Chef action — plating in kitchen | Medium shot, candid | Overhead kitchen lights | Authentic kitchen setting | Must-have |

### 3.3 Mood Board Requirements

Every shoot must have an approved mood board containing:
- 5-8 reference images showing desired style, lighting, and composition
- Color palette for the shoot (backgrounds, props, wardrobe)
- Specific "do" and "don't" examples
- Brand guidelines reference (if shooting for an existing brand)
- Client-approved before shoot day

## 5. On-Set Workflow

### 5.1 Shoot Day Timeline (Full-Day Example)

| Time | Activity | Notes |
| :--- | :--- | :--- |
| 08:00 | Arrive, unload gear, assess light | 30 min before official start |
| 08:30 | Set up first scene, test shots | Share with client for alignment |
| 09:00 | Begin shooting (priority shots first) | Work through shot list systematically |
| 10:30 | Review progress with client (if present) | Quick tethered review |
| 11:00 | Continue shooting (secondary shots) | Adjust based on client feedback |
| 12:30 | Lunch break | 45 minutes |
| 13:15 | Afternoon session (different setups/locations) | Best light for interiors: 14:00-16:00 |
| 15:30 | Final shots, creative experiments | Try unexpected angles, details |
| 16:30 | Wrap, pack gear, quick card backup | Backup to laptop before leaving |
| 17:00 | Depart | Thank host/client |

### 5.2 On-Set Communication

| Situation | Protocol |
| :--- | :--- |
| Client present on set | Show key shots on tethered screen every 30 min |
| Client not present | Send 3-5 preview shots via WhatsApp at midday |
| Creative disagreement | Shoot the client's version AND your version (present both later) |
| Technical issue (gear failure) | Switch to backup immediately, inform client only if it affects timeline |
| Weather change (outdoor shoot) | Have indoor backup plan ready, or reschedule within 48 hours |

## 6. Post-Production

### 6.1 Editing Workflow

| Step | Duration | Tool | Responsible |
| :--- | :--- | :--- | :--- |
| Import and backup | Same day | Lightroom + external drive | Photographer |
| Culling (select best shots) | Day 1-2 | Lightroom (flags + stars) | Photographer |
| Color correction (batch) | Day 2-3 | Lightroom (presets + manual) | Photographer |
| Retouching (hero shots) | Day 3-5 | Photoshop (skin, cleanup, compositing) | Photographer or Retoucher |
| Client gallery preparation | Day 5 | Lightroom export + gallery platform | Photographer |
| Client review and selection | Day 5-8 | Online gallery (password-protected) | Client |
| Final edits (selected images) | Day 8-10 | Lightroom + Photoshop | Photographer |
| Delivery | Day 10-12 | Google Drive / Dropbox | Studio Coordinator |

### 6.2 Editing Standards

| Element | Standard |
| :--- | :--- |
| White balance | Consistent across all images in a set |
| Exposure | Even, no blown highlights or crushed shadows |
| Color grading | Consistent preset/look across the set, matches brand |
| Skin retouching | Natural (remove blemishes, keep texture and pores) |
| Object removal | Clean backgrounds, remove distractions |
| Cropping | Consistent aspect ratios within a set |
| Sharpening | Output-appropriate (web vs. print) |
| File format | JPG (quality 90-95%) for delivery, RAW archived |

### 6.3 Delivery Specifications

| Use Case | Format | Resolution | Color Space |
| :--- | :--- | :--- | :--- |
| Web/social media | JPG | 2000px longest edge | sRGB |
| Print (brochure, menu) | TIFF or JPG | Full resolution (6000px+) | Adobe RGB or CMYK |
| Billboard/large format | TIFF | Full resolution | CMYK (printer profile) |
| Client archive | JPG (high quality) | Full resolution | sRGB |

## 7. Client Gallery & Selection

### 7.1 Gallery Presentation

| Element | Standard |
| :--- | :--- |
| Platform | Password-protected online gallery (Pixieset or similar) |
| Images shown | All "selects" from culling (typically 2-3x final deliverable count) |
| Editing level | Color-corrected, not fully retouched |
| Client instructions | "Please select your top [X] images for full retouching and delivery" |
| Selection deadline | 5 business days |
| Additional selections | Available at per-image rate (150-300 MAD/image) |
