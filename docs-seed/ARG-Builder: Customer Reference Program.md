# ARG-Builder: Customer Reference Program

## Executive Summary

This document defines ARG-Builder's Customer Reference Program — a structured system for identifying, nurturing, and activating customer advocates who accelerate sales cycles, strengthen brand credibility, and drive organic growth. A well-managed reference program is one of the highest-ROI investments a B2B SaaS company can make, directly reducing sales cycle length by 30–50% and increasing close rates by 20–40%.

### Reference Call Best Practices

| Phase | Duration | Content | Tips |
|-------|----------|---------|------|
| Prep (before call) | 10 min | Brief reference on prospect's situation, questions likely asked | Don't over-script; authenticity matters |
| Introduction | 2 min | How they found ARG-Builder, their role | Let reference lead naturally |
| Challenge | 5 min | What problem they were solving | Encourage specifics, not generalities |
| Solution | 5 min | How ARG-Builder helped, implementation experience | Real stories > feature lists |
| Results | 5 min | Measurable outcomes, ROI | Numbers and timeframes |
| Q&A | 10 min | Prospect's specific questions | Honest answers build trust |
| Wrap-up | 3 min | Any final thoughts, next steps | Thank the reference warmly |

## Program Operations

### Reference Request Workflow

| Step | Action | SLA | Owner |
|------|--------|-----|-------|
| 1 | Sales rep submits reference request (CRM form) | — | AE/SE |
| 2 | Reference Manager reviews and matches | < 4 hours | Reference Manager |
| 3 | Reference contacted and confirmed | < 24 hours | Reference Manager |
| 4 | Briefing materials sent to reference | < 24 hours | Reference Manager |
| 5 | Call scheduled between prospect and reference | < 48 hours | Reference Manager |
| 6 | Post-call follow-up and thank you | Same day | Reference Manager |
| 7 | Feedback collected from both parties | < 3 days | Reference Manager |

### Matching Criteria

| Factor | Weight | Rationale |
|--------|--------|-----------|
| Industry match | 30% | Prospects trust peers in same industry |
| Company size match | 25% | Similar scale = similar challenges |
| Use case match | 20% | Relevant experience with same features |
| Persona match (title/role) | 15% | Peer-to-peer conversations are most effective |
| Geography/culture | 10% | Regional considerations for enterprise |
