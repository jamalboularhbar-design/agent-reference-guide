# ArtKech Design Studio — Freelancer & Contractor Management

## 1. Purpose

ArtKech maintains a network of vetted freelancers to handle overflow work, specialized skills (3D, motion, illustration), and seasonal peaks. This SOP defines how we source, vet, onboard, manage, and pay freelance contractors while maintaining our quality standards and protecting client relationships.

## 2. When to Engage Freelancers

| Trigger | Decision | Approval |
| :--- | :--- | :--- |
| Team at > 90% capacity for 2+ weeks | Engage freelancer for overflow | Studio Coordinator |
| Specialized skill needed (3D, animation, illustration) | Engage specialist freelancer | Creative Director |
| Rush project with tight deadline | Engage freelancer to parallelize work | Creative Director |
| Team member on leave (> 1 week) | Assess if coverage needed | Studio Coordinator |
| Large project requiring multiple designers | Augment team temporarily | Creative Director |
| Client requests specific style we don't offer in-house | Subcontract to specialist | Creative Director + Account Manager |

## 4. Vetting Process for New Freelancers

### 4.1 Sourcing Channels

| Channel | Quality | Volume |
| :--- | :--- | :--- |
| Referrals from existing network | High | Low |
| Behance / Dribbble portfolio review | Medium-High | Medium |
| Local design community (events, meetups) | High | Low |
| LinkedIn outreach | Medium | Medium |
| Design school graduates (ESAV, Sup de Co) | Variable | High |

### 4.2 Vetting Steps

| Step | Duration | Assessed By |
| :--- | :--- | :--- |
| 1. Portfolio review (online) | 15 min | Creative Director |
| 2. Initial call (chemistry, availability, rates) | 30 min | Studio Coordinator |
| 3. Test project (paid, small scope) | 2-5 days | Creative Director evaluates output |
| 4. Reference check (2 previous clients) | 1-2 days | Studio Coordinator |
| 5. Contract signing + onboarding | 1 day | Studio Coordinator |

### 4.3 Test Project Guidelines

- Always paid (at their quoted rate)
- Small scope: 4-8 hours of work maximum
- Real project (not a hypothetical exercise)
- Evaluated on: quality, communication, deadline adherence, file organization
- Pass/fail decision within 48 hours of delivery

## 5. Onboarding a New Freelancer

### 5.1 Onboarding Checklist

| Item | Responsible | Timeline |
| :--- | :--- | :--- |
| Contract signed (NDA + service agreement) | Studio Coordinator | Before first project |
| Payment details collected (bank/PayPal) | Studio Coordinator | Before first project |
| Added to Slack (freelancer channel) | Studio Coordinator | Day 1 |
| Brand guidelines shared (ArtKech standards) | Creative Director | Day 1 |
| File naming conventions shared | Studio Coordinator | Day 1 |
| Toggl Track access (if time-based billing) | Studio Coordinator | Day 1 |
| Google Drive access (project folder only) | Studio Coordinator | Per project |
| Introduction to Account Manager (if client-adjacent) | Account Manager | First project |

### 5.2 What Freelancers Receive

- ArtKech brand guidelines (for understanding our quality bar)
- File delivery standards document (naming, formats, organization)
- Communication expectations (response times, availability windows)
- Project brief (same format as internal briefs)
- Access to relevant project assets and references

### 5.3 What Freelancers Do NOT Receive

- Direct client contact information (all communication goes through ArtKech)
- Full client list or studio financials
- Access to other clients' projects or files
- Internal pricing or margin information
- Admin access to any studio tools

## 6. Project Assignment & Management

### 6.1 Assignment Process

| Step | Action | Responsible |
| :--- | :--- | :--- |
| 1 | Identify need (overflow, specialty, rush) | Studio Coordinator |
| 2 | Check roster availability | Studio Coordinator |
| 3 | Brief freelancer (same format as internal) | Account Manager or Creative Director |
| 4 | Confirm timeline and rate | Studio Coordinator |
| 5 | Share project assets and access | Studio Coordinator |
| 6 | Set check-in schedule (daily for rush, 2x/week for standard) | Creative Director |

### 6.2 Communication Protocol

| Scenario | Channel | Frequency |
| :--- | :--- | :--- |
| Project updates | Slack (freelancer channel) | Daily or as agreed |
| File delivery | Google Drive (shared folder) | Per milestone |
| Feedback on work | Figma comments or Slack thread | Within 4 hours of delivery |
| Urgent issues | WhatsApp (direct to Creative Director) | As needed |
| Scheduling/availability | Email | 48 hours advance notice |

### 6.3 Quality Control

| Checkpoint | Standard | Action if Below |
| :--- | :--- | :--- |
| First draft delivery | Meets brief requirements, acceptable quality | Detailed feedback, allow 1 revision |
| Second draft | Incorporates feedback, near-final quality | Minor tweaks only, or reassign if still off |
| Final delivery | Production-ready, matches ArtKech standards | Accept and integrate, or reject and redo internally |

**Rule:** All freelancer work is reviewed by Creative Director or Senior Designer before client presentation. Freelancer work is never sent directly to clients.

## 7. Payment & Contracts

### 7.1 Payment Terms

| Arrangement | Payment Schedule | Invoice Process |
| :--- | :--- | :--- |
| Per-project (fixed fee) | 50% on assignment, 50% on approved delivery | Freelancer invoices ArtKech |
| Hourly (time-tracked) | Bi-weekly or monthly (based on Toggl hours) | Freelancer submits timesheet + invoice |
| Retainer (ongoing) | Monthly, paid by 5th of following month | Standing invoice, adjusted for actual hours |

### 7.2 Rate Negotiation

| Freelancer Experience | Acceptable Rate Range (MAD/hour) |
| :--- | :--- |
| Junior (< 2 years) | 150 – 250 |
| Mid-level (2-5 years) | 250 – 400 |
| Senior (5+ years) | 400 – 600 |
| Specialist (rare skill) | 500 – 800 |

**Markup to client:** Freelancer cost is marked up 40-60% when billed to client (covers management, QC, risk, and coordination overhead).

### 7.3 Contract Essentials

Every freelancer contract includes:
- Scope of work and deliverables
- Timeline and milestones
- Payment terms and rate
- Intellectual property assignment (all work belongs to ArtKech/client upon payment)
- Non-disclosure agreement (client information is confidential)
- Non-solicitation (cannot approach ArtKech clients directly for 12 months)
- Termination clause (either party, 7 days notice for ongoing work)
- Quality standards and revision expectations
