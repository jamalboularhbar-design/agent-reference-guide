# ArtKech Design Studio — Social Media Content Pipeline

## 1. Purpose

Social media content management is one of ArtKech's recurring revenue streams. We manage visual content creation for clients on monthly retainers, producing Instagram posts, Stories, Reels covers, LinkedIn graphics, and Facebook content. This SOP defines the end-to-end pipeline from content planning through publishing and performance reporting.

## 4. Content Planning

### 4.1 Content Pillars (Defined Per Client)

Each client has 4-6 content pillars that rotate throughout the month:

| Pillar Type | Purpose | Example (Restaurant Client) | Frequency |
| :--- | :--- | :--- | :--- |
| Product/Service showcase | Drive sales/bookings | Dish photography, menu highlights | 40% of posts |
| Behind-the-scenes | Build authenticity | Kitchen prep, team moments, sourcing | 20% of posts |
| Educational/Value | Establish expertise | Food tips, ingredient stories, recipes | 15% of posts |
| Social proof | Build trust | Customer reviews, UGC reposts, press mentions | 15% of posts |
| Community/Culture | Build connection | Local events, holidays, team celebrations | 10% of posts |

### 4.2 Content Calendar Template

| Date | Platform | Pillar | Format | Visual Description | Caption Theme | Hashtag Set |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| Jun 2 | IG Feed | Product | Carousel (5 slides) | New summer menu items, flat lay | "Summer flavors have arrived" | #MarrakechFood #SummerMenu |
| Jun 3 | IG Story | BTS | Video (15s) | Chef preparing signature dish | Poll: "Which sauce do you prefer?" | — |
| Jun 4 | IG Feed | Social Proof | Single image | Customer photo (repost with permission) | Thank you + story about the customer | #CustomerLove |

### 4.3 Calendar Approval Process

1. Account Manager sends content calendar to client by Day 3 of the month
2. Client has 48 hours to review, comment, and approve
3. Comments are addressed within 24 hours
4. If no response within 48 hours: gentle reminder sent
5. If no response within 72 hours: calendar is considered approved (stated in contract)
6. Changes after approval: accepted for up to 20% of posts, additional changes quoted separately

| LinkedIn Carousel (PDF) | Document | 1080 × 1080 px | PDF (multi-page) |
| Twitter/X | Landscape | 1600 × 900 px | JPG/PNG |

### 5.2 Design Quality Standards

| Element | Standard |
| :--- | :--- |
| Brand consistency | Every post uses client's brand colors, fonts, and visual language |
| Typography | Maximum 2 typefaces per post, readable at mobile size (minimum 24pt for body) |
| Image quality | No pixelation, proper white balance, consistent filter/preset |
| Text overlay | Maximum 20% of image area covered by text (Instagram algorithm preference) |
| Logo placement | Subtle watermark or logo in consistent position (bottom-right default) |
| Carousel flow | Clear visual progression, consistent layout across slides, CTA on last slide |
| Accessibility | Alt text prepared for every image, sufficient color contrast |

### 6.3 Real-Time Content (Stories)

Stories are not pre-scheduled — they're created and posted same-day:
- Studio Coordinator posts Stories according to the calendar theme
- Spontaneous Stories (events, trending topics) require Account Manager approval via WhatsApp
- Story templates are pre-designed (5-8 templates per client) for quick deployment
- Stories are saved to Highlights according to the client's highlight strategy

## 7. Community Management

### 7.1 Response Standards (If Included in Package)

| Interaction Type | Response Time | Tone |
| :--- | :--- | :--- |
| Comment (positive) | Within 4 hours | Warm, grateful, personal |
| Comment (question) | Within 2 hours | Helpful, informative, redirect to DM if complex |
| Comment (negative) | Within 1 hour | Empathetic, take offline ("Please DM us so we can help") |
| DM (inquiry) | Within 2 hours | Professional, helpful, convert to action |
| DM (complaint) | Within 1 hour | Apologetic, escalate to client if needed |
| Mention/tag | Within 4 hours | Acknowledge, repost if appropriate |

### 7.2 Escalation Protocol

| Situation | Action |
| :--- | :--- |
| Negative review/comment about product quality | Screenshot, notify client immediately, do NOT respond without client approval |
| Spam/inappropriate comments | Hide or delete, block repeat offenders |
| Influencer/press inquiry | Forward to client's PR contact or Account Manager |
| Crisis (viral negative content) | Immediate call to client, pause all scheduled posts, await instructions |
