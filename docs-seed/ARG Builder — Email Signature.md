## Setup Instructions

**Gmail:** Go to Settings → See all settings → General → Signature → Create new → Paste the HTML version (use "Insert as HTML" via a Chrome extension like "HTML Email Signature" or paste into the rich text editor).

**Outlook:** Go to File → Options → Mail → Signatures → New → Paste the HTML version into the editor.

**Apple Mail:** Create the signature in a rich text editor first, then paste into Mail → Preferences → Signatures.

Replace the bracketed placeholders with your actual headshot URL, Cal.com booking link, and LinkedIn profile URL before activating.
