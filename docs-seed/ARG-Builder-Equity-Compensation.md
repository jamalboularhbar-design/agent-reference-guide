# ARG-Builder: Employee Equity & Compensation Philosophy

## Complete Framework for Fair, Transparent, and Competitive Total Compensation

---

## 1. Executive Summary

Compensation is the most tangible expression of how a company values its people. In a competitive talent market, getting compensation wrong means losing top performers to companies that get it right. This document defines ARG-Builder's total compensation philosophy — covering base salary, equity, bonuses, and benefits — with the transparency, fairness, and competitiveness needed to attract and retain exceptional talent through every growth stage.

## 2. Compensation Philosophy

### 2.1 Core Principles

| Principle | Definition | Implementation |
|-----------|-----------|---------------|
| Pay for impact | Compensation reflects value created, not tenure | Level-based bands tied to scope and outcomes |
| Market competitive | Competitive enough to attract top talent | Target 65th–75th percentile for total comp |
| Internal equity | Fair relative to peers at same level | Consistent bands, regular equity audits |
| Transparency | Employees understand how comp is determined | Published bands, clear criteria |
| Simplicity | Easy to understand and administer | Fewer components, clear formulas |
| Long-term alignment | Equity creates shared ownership mentality | Meaningful equity grants for all employees |

### 2.2 Total Compensation Components

| Component | Purpose | % of Total (IC) | % of Total (Exec) |
|-----------|---------|-----------------|-------------------|
| Base salary | Predictable income, market competitiveness | 60–70% | 40–50% |
| Equity (stock options/RSUs) | Long-term alignment, wealth creation | 20–30% | 35–45% |
| Variable/bonus | Performance incentive, goal alignment | 0–10% | 10–20% |
| Benefits | Health, wellness, security | ~$15K–$25K value | ~$15K–$25K value |

## 9. Offer Construction

### 9.1 New Hire Offer Framework

| Component | Determination Method |
|-----------|---------------------|
| Level | Interview assessment → level recommendation |
| Base salary | Band for level × geographic adjustment |
| Equity | Grant guidelines for level × stage |
| Signing bonus (if applicable) | Forfeited equity/bonus at previous employer |
| Start date | Candidate preference (within reason) |
| Benefits | Standard package (same for all) |

### 9.2 Offer Negotiation Guidelines

| Candidate Request | Response |
|------------------|----------|
| Higher base salary | If within band: consider. If above band: discuss level or decline |
| More equity | Preferred lever — can increase equity more flexibly than cash |
| Signing bonus | Acceptable if replacing forfeited comp (with 1-year clawback) |
| Different title | If title matches level: flexible. If inflated: decline |
| Remote/hybrid flexibility | Policy-based, not comp-based |
| Earlier start date | Accommodate if possible |
| Relocation assistance | Standard package available |

### 9.3 Competing Offer Response

| Situation | Response |
|-----------|----------|
| Candidate has higher competing offer | Understand total comp (not just base), articulate our equity value |
| Candidate has FAANG offer | Acknowledge gap, emphasize equity upside + impact + growth |
| Candidate leveraging multiple offers | Move quickly, be transparent about our best offer |
| Candidate requests match | We don't match — we pay fairly within our bands |
| Candidate is above our band | Honest conversation — may not be right fit at this stage |
